# num_methods

A repository of academic Matlab scripts ranging various areas of numerical methods. It aims to provide clean and understandable code, more tailored towards academic applications rather than production code.

Most programs work on base MatLab, but certain ones (gaussian cuadrature) require the symbolic math toolbox. All programs are documented with docstrings, so typing `help foo` for any one of the program's names will show their syntax and arguments. Make sure to have the files in the current MatLab working directory (or one added to the path) for them to work.

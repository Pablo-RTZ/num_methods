# ODE

## Introduction

This section documents the initial value problem (IVP) solvers included in the repository. They work for both linear and non-linear ODE. The goal is to describe the numerical methods, their implementation style, and the expected convergence order for each. It is not intended to be a full numerical analysis reference; proofs are omitted and the emphasis is on the algorithmic structure.

The methods in this repository solve an ODE of the form

$$y'(t) = f(t, y(t)), \qquad y(t_0) = y_0$$

where $y$ may be a vector when the ODE system has order higher than one. All routines use a uniform mesh of $n$ steps on $[a,b]$ and return the nodes $X$ and the approximated solution values $Y$.

The implementations prioritize clarity and readability over performance. The initial step for multistep methods is usually generated by a one-step method.

## Euler

Euler's method is the simplest one-step explicit integrator. Given a step size $h$, the iteration is

$$y_{k+1} = y_k + h \, f(t_k, y_k).$$

This method is first-order accurate. The global error behaves like $O(h)$, so the order of convergence is 1.

## Heun

Heun's method is an explicit second-order Runge-Kutta integrator, also known as the improved Euler or explicit trapezoidal method. It computes

$$k_1 = f(t_k, y_k),$$
$$k_2 = f(t_k + h, y_k + h k_1),$$
$$y_{k+1} = y_k + \frac{h}{2}(k_1 + k_2).$$

This method has order of convergence 2.

## RungeKutta

This `RungeKutta` implementation corresponds to the classical fourth-order Runge-Kutta method (RK4). The scheme is:

$$k_1 = f(t_k, y_k),$$
$$k_2 = f(t_k + \frac{h}{2}, y_k + \frac{h}{2} k_1),$$
$$k_3 = f(t_k + \frac{h}{2}, y_k + \frac{h}{2} k_2),$$
$$k_4 = f(t_k + h, y_k + h k_3),$$
$$y_{k+1} = y_k + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4).$$

RK4 has order of convergence 4.

## AdamsBashforth

Adams-Bashforth is a linear multistep method. The implementation in this repository uses the two-step Adams-Bashforth formula:

$$y_{k+1} = y_k + \frac{h}{2} \bigl(3 f(t_k, y_k) - f(t_{k-1}, y_{k-1}) \bigr).$$

A one-step predictor (Heun) is used to generate the first value. The global order of convergence of this two-step Adams-Bashforth method is 2.

## Milne method

Milne's method is a predictor-corrector multistep algorithm. The predictor is an explicit Milne formula and the corrector is an implicit one based on a chosen quadrature rule.

The predictor used is:

$$y_{k+1}^{(p)} = y_{k-3} + \frac{4h}{3} \bigl(2f(t_k, y_k) - f(t_{k-1}, y_{k-1}) + 2f(t_{k-2}, y_{k-2})\bigr).$$

This predictor itself is of order 4.

### Milne-Simpson

In `MilneSimpson`, the implicit corrector is Simpson's rule:

$$y_{k+1} = y_{k-1} + \frac{h}{3} \bigl( f(t_{k+1}, y_{k+1}) + 4 f(t_k, y_k) + f(t_{k-1}, y_{k-1}) \bigr).$$

The method is implemented by using the explicit Milne predictor to compute an initial guess $y_{k+1}^{(p)}$, then evaluating the corrector with that guess. The combination yields a fourth-order convergence method, order 4.

### Milne-Trap

The `MilneTrap` implementation uses the implicit trapezoidal rule as the corrector:

$$y_{k+1} = y_k + \frac{h}{2} \bigl( f(t_k, y_k) + f(t_{k+1}, y_{k+1}) \bigr).$$

Again, the explicit Milne predictor provides an initial approximation for $y_{k+1}$, and the implicit integrator is applied with that approximation. The resulting predictor-corrector scheme also achieves order 4 in practice for smooth problems.

### Adapting Milne to other implicit integrators

Milne's predictor step can be paired with any implicit corrector that defines $y_{k+1}$ through an integral approximation involving $f(t_{k+1}, y_{k+1})$. The general strategy is:

1. Use the explicit Milne predictor to compute $y_{k+1}^{(p)}$.
2. Substitute $y_{k+1}^{(p)}$ into the right-hand side of the chosen implicit formula.
3. Use the result as the corrected value $y_{k+1}$, or iterate the corrector if a more accurate solution is desired.

Examples:

- Simpson corrector: uses the implicit Simpson quadrature rule and requires $f(t_{k+1}, y_{k+1})$.
- Trapezoidal corrector: uses the implicit trapezoidal formula with $f(t_{k+1}, y_{k+1})$.

This approach is not restricted to these two rules; any implicit integrator that is consistent and stable with respect to the step size can be used as the corrector.
